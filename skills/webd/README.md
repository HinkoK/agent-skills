# webd

A web-design skill for landing pages, SaaS frontends, conversion pages, and non-generic UI.

## What it does

Helps the agent choose an aesthetic direction, conversion strategy, and UI/component libraries before coding.

Includes:
- design framework;
- conversion patterns;
- shadcn/ui-based library references.

## Install

```bash
hermes skills install https://raw.githubusercontent.com/HinkoK/agent-skills/main/skills/webd/SKILL.md
```

For the full experience, clone this repo too so the skill can access `references/`.

## Example

```text
/webd design a premium landing page for an AI assistant with a strong hero and clear CTA
```
