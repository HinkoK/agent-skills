# aftercall

Post-call analysis skill for recordings, transcripts, meeting notes, therapy/coaching sessions, study calls, and voice notes.

## What it does

Turns conversations into useful output:

- transcript or cleaned transcript;
- structured summary;
- decisions;
- action items and owners;
- follow-up message;
- study notes;
- reflection prompts;
- optional speaker-aware transcript.

## Install

```bash
hermes skills install https://raw.githubusercontent.com/HinkoK/agent-skills/main/skills/aftercall/SKILL.md
```

## Example

```text
Разбери этот созвон: сделай summary, decisions, action items и follow-up сообщение.
```
