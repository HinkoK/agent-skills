# Export templates

## Telegram summary

```md
Итог созвона:
- Что решили: ...
- Кто делает: ...
- Следующий шаг: ...
- Что осталось открытым: ...
```

Use when the result should be fast to scan in chat.

## Obsidian note

```md
# After-call: [title]

Дата: [YYYY-MM-DD]
Тип: [work-call / therapy-session / client-follow-up / study-call]

## Коротко
[1-2 sentence recap]

## Что обсудили
- ...

## К чему пришли
- ...

## Что делать дальше
- [ ] ...

## Открытые вопросы
- ...

## Теги
#aftercall #[mode]

## Связано
- [[Related Note 1]]
- [[Related Note 2]]
```

Use when the user wants a durable note or second-brain entry.

## Task list

```md
# Tasks from call

- [ ] Task 1
  - Owner: ...
  - Deadline: ...

- [ ] Task 2
  - Owner: ...
  - Deadline: ...
```

Use when the user mainly needs execution clarity.

## Speaker transcript export

```md
Участники:
- Speaker 1 = ...
- Speaker 2 = ...

[Speaker 1]
...

[Speaker 2]
...
```

Use when the user wants a readable transcript with speaker separation.
