# Output templates

## 1) Standard work call

```md
Коротко:
[1-2 sentence summary]

Что обсудили:
- ...
- ...

К чему пришли:
- ...
- ...

Кто за что отвечает:
- [Имя] — ...
- [Имя] — ...

Что делать дальше:
- [ ] ...
- [ ] ...

Открытые вопросы:
- ...
```

## 2) Personal / therapy / coaching session

```md
Коротко:
[1-2 sentence summary]

Главные темы:
- ...
- ...

Что особенно всплыло:
- ...
- ...

На что можно обратить внимание между сессиями:
- ...
- ...

Практический следующий шаг:
- ...

Осталось неясным:
- ...
```

## 3) Client follow-up

```md
Коротко:
[1-2 sentence summary]

Что подтвердили:
- ...

Следующие шаги:
- ...

Draft follow-up:
Привет! Спасибо за созвон.

Кратко фиксирую, к чему пришли:
- ...
- ...

Следующие шаги:
- ...
- ...

Если всё ок, двигаемся так.
```

## 5) Speaker-labeled transcript

```md
Участники:
- Speaker 1 = [optional name/role]
- Speaker 2 = [optional name/role]

Транскрипт:

[Speaker 1]
...

[Speaker 2]
...

Коротко по ролям:
- Speaker 1: ...
- Speaker 2: ...
```
```

Use when attribution matters more than a flat transcript.

## 6) Ultra-short mode

```md
Итог созвона:
- Решили: ...
- Делает: ...
- Следующий шаг: ...
- Открытый вопрос: ...
```
